# DevShowcase

**DevShowcase** is a modern platform for developers to showcase their projects, share their work, and discover projects from other developers in the community.

## ✨ Features

### 🎨 For Developers
- **Portfolio Platform**: Showcase your best projects with beautiful project cards
- **Project Details**: Detailed project pages with descriptions, tech stack, and pricing info
- **Social Features**: Connect with other developers through direct messaging
- **Reviews & Ratings**: Rate and review projects from the community
- **User Profiles**: Create a professional profile highlighting your work and achievements
- **Notifications**: Stay updated with likes, reviews, and messages
- **Search & Discovery**: Find projects by technology, category, or trending topics

### 🔐 Authentication
- Secure login and registration system
- User account management
- Profile customization

### 💬 Community
- Direct messaging between developers
- Project reviews and ratings
- Trending projects feed
- Real-time notifications

## 🛠️ Tech Stack

- **Frontend**: Vanilla JavaScript with modern CSS
- **Build Tool**: Vite
- **Backend**: Supabase (Authentication & Database)
- **Fonts**: Syne, Instrument Sans, JetBrains Mono
- **Deployment**: Vercel
- **Package Manager**: npm

## 🎨 Design

- **Color Scheme**: Dark theme with lime green accents (#b8f53e)
- **Dark Background**: #0a0a0a
- **Light Text**: #f4f0e8
- **Accent Colors**: Blue (#4fa3e8), Red (#e85b4f), Amber (#f0a832)
- **Responsive**: Mobile-friendly design

## 🚀 Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/emiliano0990/ShowbitHub.git

# Navigate to project directory
cd ShowbitHub

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
